## プロジェクト概要

コンビニ店舗の**事務所内金庫の現金**を管理するアプリ。

想定する機能の軸:

- 金種別（10000 / 5000 / 1000 / 500 / 100 / 50 / 10 / 5 / 1 円）の枚数管理
- 入出金の記録（釣銭準備金の払い出し、売上金の入金・回収 など）
- 残高照合と過不足チェック（帳簿残高 vs 実査カウントの突き合わせ）

**現状はコード未着手**（このファイル以外に何も存在しない）。実装を開始したら、実際のファイル構成・データモデルに合わせてこのファイルを更新すること。

## 技術方針（ユーザーと合意済みの制約）

- 単一の `index.html` に HTML / CSS / JS をすべてインラインで書く。ビルドツール・外部依存・CDN は使わない
- データ永続化は `localStorage`（JSON）。サーバー不要
- UI は日本語
- 金額は**円の整数**で扱う（浮動小数点で金額計算をしない）

## 実行方法

```bash
open index.html
```

ブラウザで開くだけ。ビルド・テスト・lint の仕組みは存在しない。
